Keypresses and Gestures:
Straight Line Gestures
- swipe left gesture + left arrow key - Jumps back 3 seconds
- swipe right gesture + right arrow key - Jumps forwards 3 seconds
- swipe up gesture + up arrow key - Increases WPM by 50
- swipe down gesture + down arrow key - Decreases WPM by 50

Angle Gestures 
- ^ gesture + "+" or "=" key - Increases the font by 8
- v gesture + "-" key - Decreases the font by 8
- > gesture + spacebar key - Toggles the pause/play button
